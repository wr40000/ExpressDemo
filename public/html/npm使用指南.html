npm 查看 npm 全局安装的包，是否安装了某个包

查看全局安装的包

npm list -g --depth 0

-g:全局的安装包

list：已安装的node包

–depth 0：列表搜索深度0级 不加这参数 会默认所有 npm list -g

npm list -g --depth 1

查看全局是否安装过某个包 npm ls 包/插件名称 -g

查看本地/当前文件下是否安装某个包（不加-g） npm ls 包/插件名称

下载express包

npm install express --save

查看全局安装路径

npm root -g

查看npm的基础设置

npm config ls

查看安装目录路径

npm config get prefix

json-server启动命令

json-server --watch db.json

npm 怎么查看一个包的历史版本

要查看一个npm包的历史版本，可以使用以下命令：

复制代码npm view <package-name> versions

其中，将<package-name>替换为你想要查询的npm包的名称。这个命令会显示出该npm包的所有发布版本号。

你也可以在输出中加入--json选项，以JSON格式的方式获取版本信息，例如：

复制代码npm view <package-name> versions --json

无论是哪种方式，都能够让你获取该npm包历史版本的完整列表。
