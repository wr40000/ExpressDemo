MPV播放器安装、配置以及使用演示视频

mpv是一个自由开源的跨平台媒体播放器，具有轻量、兼容性好、播放性能优秀等特点

mpv官网：https://mpv.io

Windows版下载地址：https://sourceforge.net/projects/mpv-player-windows/files/

脚本介绍、网盘分享：https://www.bilibili.com/read/cv15690307

mpv的默认mpv.conf设置为最大兼容性而定，因而默认软解，这里将之改为硬解，并对一些其他常用选项做出修改。mpv有其一套默认快捷键设置，但不符合个人习惯，也难以记忆。在阅读官方文档、参考网友配置方法后编写了以下input.conf快捷键方案。所有内容都可以根据读者需求自行修改。

使用方法：复制以下内容到一个txt文件中，分别改名称和后缀为mpv.conf、input.conf，Windows系统，在mpv.exe所在目录创建一个portable_config文件夹置于其内（高优先级、绿色便携化，推荐），或放到%appdata%/mpv文件夹中（全局默认位置）；类Unix平台（Linux/macOS等）则放在/usr/local/etc/mpv/或~/.config/mpv/（注意，.开头的文件夹是隐藏的）。

portable_config文件夹 地址栏输入%appdata%/mpv，回车即可 建议使用支持语法高亮的文本编辑器，如开源的notepad3，相比记事本显示效果更佳。

gpu-api=auto

在Windows上，你可以使用命令行工具来查看特定端口是否被占用，但要注意，通过端口查找到具体的进程和相关文件比较复杂。以下是一些基本的方法：

查看端口占用情况：

  1. 使用命令行（Command Prompt）:

    打开命令行窗口，然后运行以下命令：

    netstat -ano | find "3000"

    如果端口3000被占用，你将看到相关的信息，包括进程的ID。你可以使用任务管理器（Task Manager）来找到相应ID的进程。

  2. 使用 PowerShell:

    打开 PowerShell 窗口，运行以下命令：

    Get-NetTCPConnection -LocalPort 3000

    这会列出所有使用3000端口的网络连接。

  3. 使用资源监视器:

    在资源监视器中，你可以查看网络活动，包括使用的端口。打开资源监视器，切换到“网络”选项卡，然后找到“监听端口”列。

    screen -dmS Ter bash -c "sh scripts/dkd+fitFn_res56_res20.sh"

查找进程和相关文件：

  1. 使用任务管理器（Task Manager）:

    打开任务管理器，切换到“详细信息”选项卡，找到占用3000端口的进程。右键点击该进程，选择“打开文件位置”来查看相关文件。

  2. 使用命令行（Command Prompt）:

    在命令行窗口中，可以使用 tasklist 命令查看所有运行的进程，然后通过进程ID使用 taskkill 命令终止进程。

    tasklist | find "3000"

    通过 taskkill 终止进程：

    taskkill /F /PID <进程ID>

请注意，这些命令可能需要以管理员身份运行，以便访问某些系统信息。通过这些方法，你可以找到占用3000端口的进程，但找到相关文件可能需要更详细的调查，例如使用文件资源管理器（Windows Explorer）的搜索功能。

删除文件 / 删除文件夹

删除文件

在命令提示符中使用 del：

del 文件名

例如，要删除名为 example.txt 的文件：

del example.txt

你可以使用通配符 * 来匹配多个文件。例如，要删除当前目录中所有以 .txt 结尾的文件：

del *.txt

在 PowerShell 中使用 Remove-Item：

Remove-Item 文件名

例如，删除名为 example.txt 的文件：

Remove-Item example.txt

同样，你也可以使用通配符 * 来匹配多个文件：

Remove-Item *.txt

请注意，删除文件是一个不可逆的操作，删除后文件将无法恢复。确保你真正想要删除这些文件，以免造成数据丢失。

删除文件夹

在命令提示符中使用 rmdir：

bashCopy code
rmdir /s /q 文件夹路径
  • /s 参数表示删除目录及其所有子目录和文件。
  • /q 参数表示安静模式，即不要求确认删除。

例如，删除名为 myfolder 的文件夹及其所有内容：

bashCopy code
rmdir /s /q myfolder

在 PowerShell 中使用 Remove-Item：

bashCopy code
Remove-Item -Recurse -Force 文件夹路径
  • -Recurse 参数表示递归删除，即删除目录及其所有子目录和文件。
  • -Force 参数表示强制删除，即不要求确认删除。

例如，删除名为 myfolder 的文件夹及其所有内容：

bashCopy code
Remove-Item -Recurse -Force myfolder

请谨慎使用删除命令，确保你真正想要删除这些文件夹及其内容，以免造成数据丢失。
